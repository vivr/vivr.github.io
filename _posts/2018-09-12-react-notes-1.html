What is React?

React is a javascript library used for building user interfaces. We create reusable components and these components display data as it changes over time. If you want to read about basic React concepts take a look at this guide from reactjs.org.

The DOM is the structure of HTML elements that make up a webpage. The DOM API refers to how these page elements are accessed and changed. React makes updating the DOM faster by using DOM Diffing. DOM Diffing happens when you compare the currently rendered content with the new UI changes that are about to be made. React optimizes this by making only the minimal changes necessary. With DOM Diffing Javascript objects are compared and this is much faster than writing to or reading from the actual DOM.

Set up a development environment

To begin make sure that Node is installed which can be downloaded from nodejs.org. We will also need to use the command line and a code editor.

  1. In the terminal, run: npm install -g create-react-app. Note this web app uses create-react-app version 1.4.1.
  2. Still in the terminal, navigate to your project folder and run: create-react-app .
  3. Next, install dependencies npm install react-router-dom react-fontawesome axios react-transition-group react-simple-sidenav react-slick --save. In this setup we are using the following dependencies:
  • react-router-dom - this navigational component is used when a React app has more than one view
  • react-simple-sidenav - component for side navigation
  • react-slick - carousel component
  • react-fontawesome - component for the font-awesome icon library
  • axios - library that allows us to make async requests just like AJAX
  • react-transition-group - an easy way to add transitions to our components

Install JSON Server

JSON Server is kind of a fake REST API which allows us to make requests to local host and get some content.

  1. In the terminal install JSON server globally: npm install -g json-server
  2. In the root directory of your project create a file called db.json and add some data to it. This will be used as our database. In my db.json file I have 3 different data collections which include articles, videos and brands
  3. Back in the terminal start the server npm start
  4. Next start the JSON server. Open a new terminal window and type: json-server --watch db.json --port 3004 . In my case this command has created the following links on port 3004: 
    			
			Resources
			http://localhost:3004/articles
			http://localhost:3004/videos
			http://localhost:3004/brands
			
			Home
			http://localhost:3004
    Note: instead of running json-server --watch db.json --port 3004 each time you want to start the JSON server you can instead add the following to the scripts section of your package.json file: "dev-serv": "json-server --watch db.json --port 3004",. Now all you need to type to run the JSON server in the terminal is npm run dev-serv.

Enable css modules

CSS Modules allow each React component to have it's own locally scoped CSS file which makes the development process much more manageable.

  1. If the dev server is running go to the terminal and press Ctrl+C to stop the server
  2. Next run npm run eject
  3. Next, start the development server. Type npm start
  4. In your project folder open config/webpack.config.dev.js and find the line: importLoaders: 1,. Select this line and add the following code: 
    			
			modules:true,
			importLoaders: 1,
			localIdentName: '[name]__[local]__[hash:base64:5]'
  5. Save and close the file
  6. Open config/webpack.config.prod.js and find the following the 3 lines of code: 
    			
			importLoaders: 1,
			minimize: true,
			sourceMap: shouldUseSourceMap,
  7. Edit the above block of code so that it matches the following:
    8. 			
			importLoaders: 1,
			modules:true,
			minimize: true,
			sourceMap: shouldUseSourceMap,
			localIdentName: '[name]__[local]__[hash:base64:5]'
  8. Save and close the file
  9. Whenever we edit any config files we need to restart the server so we can see changes. You can stop the server with Ctrl+C then start it again by typing npm start

  12. Inside the src/components/widgets/NewsList folder create a file called newsList.js. Open the file and add the following code: 
    		
		import React, { Component } from 'react';
		import {CSSTransition , TransitionGroup } from 'react-transition-group';
		import { Link } from 'react-router-dom';
		import axios from 'axios';
		
		import { URL } from '../../../config';
		import styles from './newsList.css';
		import Button from '../Buttons/buttons';
		import CardInfo from '../../widgets/CardInfo/cardInfo';
		
		class NewsList extends Component {
		
			state = {
				brands:[],
				items:[],
				start:this.props.start,
				end:this.props.start + this.props.amount,
				amount:this.props.amount
			}
		
			componentWillMount(){
				this.request(this.state.start,this.state.end)
			}
		
			request = (start,end) => {
				if(this.state.brands.length < 1){
					axios.get(`${URL}/brands`)
					.then( response  => {
						this.setState({
							brands:response.data
						})
					})
				}
		
				axios.get(`${URL}/articles?_start=${start}&_end=${end}`)
				.then( response => {
					this.setState({
						items:[...this.state.items,...response.data],
						start,
						end
					})
				})
			}
		
			loadMore = () => {
				let end = this.state.end + this.state.amount;
				this.request(this.state.end,end)
			}
		
			renderNews = (type) => {
				let template = null;
		
				switch(type){
					case('card'):
						template = this.state.items.map((item,i) => (
							<CSSTransition
								classNames={{
									enter:styles.newsList_wrapper,
									enterActive:styles.newsList_wrapper_enter
								}}
								timeout={500}
								key={i}
							>
								<div>
									<div className={styles.newslist_item}>
										<Link to={`/articles/${item.id}`}>
											<CardInfo brands={this.state.brands} brand={item.brand} date={item.date}/>
											<h2>{item.title}</h2>
										</Link>
									</div>
								</div>
							</CSSTransition>
							
						))
						break;
					case('cardMain'):
						template = this.state.items.map((item,i) => (
							<CSSTransition
								classNames={{
									enter:styles.newsList_wrapper,
									enterActive:styles.newsList_wrapper_enter
								}}
								timeout={500}
								key={i}
							>
								<Link to={`/articles/${item.id}`}>
									<div className={styles.flex_wrapper}>
										<div className={styles.left} 
											style={{
												background:`url('/images/articles/${item.image}')`
											}}>
											<div></div>
										</div>
										<div className={styles.right}>
											<CardInfo brands={this.state.brands} brand={item.brand} date={item.date}/>
											<h2>{item.title}</h2>
										</div>
									</div>
								</Link>
							</CSSTransition>
						))
						break;
					default:
						template = null;
				}
		
				return template;
			}
		
			render(){
				return(
					<div>
						<TransitionGroup
							component="div"
							className="list"
						>
							{ this.renderNews( this.props.type )}
						</TransitionGroup>
						<Button
							type="loadmore"
							loadMore={()=>this.loadMore()}
							cta="Load More News"
						/>
					</div>
				)
			}
		
		}
		
		export default NewsList;

	const CardInfo = (props) =>{

		import React, { Component } from 'react';
		import styles from './videosList.css';
		import axios from 'axios';
		
		import { URL } from '../../../config';
		import Button from '../Buttons/buttons';
		import VideosListTemplate from './videosListTemplate';
		
		class VideosList extends Component {
		
			state = {
				brands:[],
				videos:[], 
				start: this.props.start,
				end: this.props.start + this.props.amount,
				amount: this.props.amount
			}
		
			componentWillMount() {
				this.request(this.state.start, this.state.end)
			}
		
			request = (start,end) => {
				if(this.state.brands.length < 1){
					axios.get(`${URL}/brands`)
					.then( response => {
						this.setState({
							brands:response.data
						})
					})
				}
		
				axios.get(`${URL}/videos?_start=${start}&_end=${end}`)
				.then( response => {
					this.setState({
						videos:[...this.state.videos,...response.data],
						start,
						end
					})
				})
			}
		
			renderVideos = () => {
				let template = null;
		
				switch(this.props.type){
					case('card'):
						template = <VideosListTemplate data={this.state.videos} brands={this.state.brands}/>
						break;
					default:
						template = null
				}
				return template;
			}
		
			loadMore = () => {
				let end = this.state.end + this.state.amount;
				this.request(this.state.end, end)
			}
		
			renderButton = () => {
				return this.props.loadmore ? 
					<Button
						type="loadmore"
						loadMore={()=> this.loadMore()}
						cta="Load More Videos"
					/>
					: 
					<Button type="linkTo" cta="More videos" linkTo="/videos"/>
			}
		
			renderTitle = () => {
				return this.props.title ? 
					<h3><strong>Product</strong> Videos</h3>
				: null
			}
		
			render(){
				return(
					<div className={styles.videoList_wrapper}>
						{ this.renderTitle() }
						{ this.renderVideos() }
						{ this.renderButton() }
					</div>
				)
			}
		
		}
		
		export default VideosList;
